Significance test of NNA

\(H_0: \mathrm{School's\ of\ Tainan\ are\ randomly\ distributed.}\)
\(H_1: \mathrm{School's\ of\ Tainan\ are\ not\ randomly\ distributed.}\)
The test is carried out at 95% confidence level.

\(\mathrm{SE} = \frac{0.26136}{\sqrt{\frac{n^2}{A}}}\approx29.243\)
\(Z = \frac{r_\mathrm{obs}-r_\mathrm{exp}}{\mathrm{SE}}\approx-6.542\)

The critical values obtained from the z-distribution for \(\alpha=0.05\) (two-tailed test) are \(\pm1.96\). Therefore, the test fails and \(H_0\) is rejected. Alternatively, \(H_1\) which states the point pattern is not random is accepted. Since the resulting z score is \(-6.542<2\sigma\), indicating the difference is significant and the distribution is clustered.

K-order Nearest Neightbor Indices

School.knn <- apply(nndist(X = School.ppp, k = 1:100), 2, FUN=mean);

mean <- c();
knn <- list();
for (j in 1:1000) {
  random.points <- rpoint(424, win = Windows);
  mean[j] <- mean(nndist(X = random.points, k = 1));
  knn[[j]] <- apply(nndist(X = random.points, k = 1:100), 2, FUN = mean);
}

mean.df <- data.frame(mean);
mean.df$order <- order(mean);

plot(School.knn, type = "l", col = "red");
lines(knn[[which(mean.df$order == length(mean)*0.05)]], type = "l", col = "blue", lty = "dashed");
lines(knn[[which(mean.df$order == length(mean)*0.5)]], type = "l", col = "blue");
lines(knn[[which(mean.df$order == length(mean)*0.95)]], type = "l", col = "blue", lty = "dashed");

Significance test of K-NNI

\(H_0: 5^\mathrm{th} \mathrm{percentile\ KNNI} \leq \mathrm{KNNI}_{obs} \leq 95^\mathrm{th} \mathrm{percentile\ KNNI}\)
\(H_1: 5^\mathrm{th} \mathrm{percentile\ KNNI} \geq \mathrm{KNNI}_{obs} \mathrm{\ or\ } \mathrm{KNNI}_{obs} \geq 95^\mathrm{th} \mathrm{percentile\ KNNI}\)
The test is carried out at 95% confidence level.

According to the plot of K-NNI, one can clearly tell that the observational K-NNI does not belong to the interval between 5th/95th percentile K-NNI. Therefore, the test fails and \(H_0\) is rejected. Alternatively, \(H_1\) which states the point pattern is not random is accepted. Since the observational K-NNI lies way under the 5th percentile K-NNI, indicating the difference is significant and the distribution is clustered.

Significance test of G Function

H0: The G function plot of the distribution of the schools of Tainan lies within the envelope of G function plots of the random distributioin, indicating the schools are randomly distributed.
H1: The G function plot of the distribution of the schools of Tainan does not lie within the envelope of G function plots of the random distributioin, indicating the schools are not randomly distributed.

According to the plot, the G function plot of the schools of Tainan does not lie within the envelope. Therefore, the test fails and \(H_0\) is rejected. Alternatively, \(H_1\) which states the point pattern is not random is accepted. Since the G function plot of the schools of Tainan is located above the envelope, one can conclude the distribution is a clustered pattern.
