chainstore = function(d) {
  n <- d / 0.5;
  meanDensity <- 1:n;
  index <- 1:n;
  for (i in 1:n) {
    MIC.buffer <- gBuffer(MIC, byid = TRUE, id = MIC$ID, width = 1000 * i * 0.5);
    chainDensity <- poly.counts(MIC, MIC.buffer);
    print(mean(chainDensity));
    meanDensity[i] <- mean(chainDensity);
  }
  plot(x = index / 2, y = meanDensity, type = "l", xlab = "Distance (km)", ylab = "Mean Chain Density of McDonald's");
  points(x = index / 2, y = meanDensity, pch = 20);
  text(x = index / 2, y = meanDensity, meanDensity, pos = 4, cex = 0.5, offset = 0.25);
}

設立假設檢定如下(單尾檢定):
\(\bullet\ \mathrm{null\ hypothesis\ }H_0: \mu_A \leq \mu_B\)
\(\bullet\ \mathrm{alternative\ hypothesis\ }H_1: \mu_A > \mu_B\)

檢定的顯著水準\(\alpha = 0.05\)，對chainDensity_A, chainDensity_B做T檢定可得:

t.test(x = chainDensity_A, y = chainDensity_B, alternative = "greater");
## 
##  Welch Two Sample t-test
## 
## data:  chainDensity_A and chainDensity_B
## t = -2.2136, df = 35.903, p-value = 0.9833
## alternative hypothesis: true difference in means is greater than 0
## 95 percent confidence interval:
##  -2.283744       Inf

## mean of x mean of y 
##  4.230769  5.526316

因為\(t = -2.2136\)，rejection region \(RR = [-2.283744, \infty)\)，比較之後可得$ t RR\(，所以在顯著水準\)= 0.05\(下，接受虛無假設\)H_0$，也就是A區麥當勞的平均連鎖密度並非顯著高於B區麥當勞的平均連鎖密度。
