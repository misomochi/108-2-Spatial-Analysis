VMR test

\(H_0:\sigma^2=\lambda\)
\(H_1:\sigma^2\neq\lambda\)
The test is carried out at 95% confidence level.

\(\mathrm{VMR} = \frac{\sigma^2}{\lambda} = 17.68603 / 2.163265 \approx 8.175617\)
\(\mathrm{df} = k - 1 = 195\)
\(s_\mathrm{vmr}=\sqrt{\frac{2}{195}}\approx 0.1013\)
\(t = \frac{\mathrm{vmr}-1}{s_\mathrm{vmr}}\approx 70.85354\)

The critical values obtained from the t-distribution for \(\alpha=0.025\) (two-tailed test), and \(\mathrm{df}=195\) are \(\pm1.9722\). Therefore, the test fails and \(H_0\) is rejected. Alternatively, \(H_1\) which states the point pattern is not random is accepted. When the statistical testing is redone with \(H_1:\sigma^2>\lambda\) (pattern is clustered), the critical value from the t-distribution for \(\alpha=0.05\) (one-tail test) is 1.65271. The test also fails and \(H_0\) is rejected, and \(H_1\) which states that point pattern is clustered is accepted.
